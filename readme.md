# // SYSTEMS_ENGINEER & FULL-STACK_DEVELOPER
A high-performance, multi-theme portfolio built for the terminal aesthetic. This project features a decoupled CSS design system allowing for instant "hot-swapping" between distinct visual identities: Minimalism, Neo-Brutalism, Cyberpunk, and Nord.

## SYSTEM_FEATURES

### 1. Dynamic Theme EngineA custom-engineered state management system that swaps entire visual identities without re-rendering components.
* **SYS.MINIMAL:** A clean, high-contrast workspace for deep focus.
* **SYS.BRUTAL:** High-impact "Soft Brutalism" with bold borders and 16px structural curves.
* **SYS.CYBER:** Neon-drenched night city vibes with Orbitron typography and glows.
* **SYS.NORD:** An arctic-inspired developer palette with soft pastels and matte noise textures.

### 2. Project_Shelf
A motion-heavy UI component for showcasing full-stack projects like GameVault and Nectar. Features smooth flex-animations and context-aware coloring.

### 3. System_TelemetryA live data feed integrating:
* **GitHub Activity Log:** Visualizing commit frequency with a theme-aware SVG calendar.
* **Network Pings:** Interactive links to CodeChef (3-star) and Codeforces (Pupil) profiles using dynamic CSS drop-shadows.

## 🚀 LOCAL_INSTALLATION

Ensure you have Node.js and npm installed on your machine.
```Bash
# 1. Clone the repository
git clone https://github.com/MAX5271/portfolio-v3.git

# 2. Enter the directory
cd portfolio-v3

# 3. Install dependencies
npm install

# 4. Launch dev server (with mobile host access)
npm run dev
```
